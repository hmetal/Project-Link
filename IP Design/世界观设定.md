# 时间线
* 人类完成了新一代工业革命:人工智能普及、3D打印、核聚变反应堆和量子计算机
* 为争夺技术的控制权，爆发了第三次世界大战（以下简称WW3）
* WW3以三大超级大国瓜分世界作为终结，由于战争损失惨重，三大国签订协约裁撤军火，政府机构大量裁撤，减弱对社会的控制力
* 随着科技含量高、扩展性强的胶囊建筑的普及，以及居民便利性的需要，世界各地建造了许多超级都市【1】
* VR技术迅速提升，为了防止人类坠入虚幻之中，各国对VR的真实度标准严格限制，打击直接使用脑连接的VR技术
* 游戏故事时间点
# 关于超级都市
* 极为便利，管道运输作为交通最主要的形式
* 出现很多城邦制的都市，这些都市属于各大国都看不上的地盘，故事背景就发生在次时代的九龙城寨
# 政治
* 为了将人口更多地留在本国或本城邦，政府将税费降到最低，将许多工作外包给私人公司解决
